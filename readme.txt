FA Forever - Proxy Server
-------------------------

Handling nat correctly is sometimes/often impossible on some setup or routers.
This server allow people to connect to others through a unique proxy, making the NAT more easier as long as the client is able to connect to this server.

Pre-requisites are :

- Python 2.6/2.7 32 bits
- PySide with at least QT 4.7.2


If you want to contribute back to the project, please make a fork and create pull-Requests of your changes.
